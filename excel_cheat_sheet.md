# Excel Cheat Sheet

* Keyboard shortcuts
  * Select multiple cells in continous range.
    * Ctrl + Shift + Arrow key

* <https://www.reddit.com/r/educationalgifs/comments/dty4vu/60_seconds_to_learn_how_to_excel_at_using_excel/>

* Sum

```bash
=SUM(F2:G2)
```

* Average

```bash
=AVERAGE(H2:H10)
```

* Median

```bash
=MEDIAN(H2:H10)
```

* Max

```bash
=MAX(H2:H10)
```

* Min

```bash
=MIN(H2:H10)
```

* Std. Deviation

```bash
=STDEV.P(H2:H10)

=STDEV.S(H2:H10)
```

* Delete a named range.
  * Formulas tab > Name Manager > Select name to delete > Click Delete.

* Conditionals

  * IF

  ```bash
  =IF(D2 > 5, "TRUE", "FALSE")
  ```

  * COUNTIF
  
  ```bash
  =COUNTIF(Colors, "=Blue")
  ```
